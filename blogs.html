About java package

january 2023

you now about many java package but right now we are studying about new java package that is java.time . In jave.time package there are so many classes like LocalDate , LocalTime , LocalDateTime and DateTimeFormatter .

Displaying Current Date

To display the current date we import java.time.LocalDate class , and use now () method : Example:-
import java.time.LocalDate
class CurrentDate {
public static void main(String arg[]) {
LocalDate myObj = LocalDate.now();
System.out.println(myObj);
}
}

Displaying Current Time

To display the current time (hour, minute, second, and nanoseconds) we import java.time.LocalTime class , and use now () method :
Example:-
import java.time.LocalTime
class CurrentDate {
public static void main(String arg[]) {
LocalTime myObj = LocalTime.now();
System.out.println(myObj);
}
}

Displaying Current Date and Time

To display the current date and time both we import java.time.LocalDateTime class , and use it now() method :
Example:-
import java.time.LocalDateTime
class CurrentDateTime {
public static void main(String arg[]) {
LocalDateTime myObj = LocalDateTime.now();
System.out.println(myObj);
}
}

Formatting Date and Time

For formatting Date and Time we import java.time.format.DateTimeFormatter class , and use it ofPattern() method :
Example:-
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
public class Main {
public static void main(String[] args) {
LocalDateTime myDateObj = LocalDateTime.now();
System.out.println("Before formatting: " + myDateObj);
DateTimeFormatter myFormatObj = DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm:ss");
String formattedDate = myDateObj.format(myFormatObj);
System.out.println("After formatting: " + formattedDate);
}
}
